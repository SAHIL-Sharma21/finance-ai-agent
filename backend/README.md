# finance_agent

To install dependencies:

```bash
bun install
```
Get Api key from Groq:
```bash
GROQ_API_KEY="<Your groq api key>"
```


To run:

```bash
bun run agent.ts
```


This project was created using `bun init` in bun v1.1.45. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
